# Fraud Detection Engine

**Course 624: Fraud Analysis - Mini-Project**
Behavioral Intelligence for Academic Misconduct Detection.

A simple, end-to-end machine-learning pipeline that classifies images of
students during exams as **Normal (0)** or **Fraud (1)**, then compares seven
classifiers and recommends the best one for a university setting.

The design goal is **simple but sufficient**: the smallest, clearest pipeline
that still satisfies every project requirement. No deep-learning training, no
gradient-boosting libraries - just scikit-learn and a handful of standard
packages.
